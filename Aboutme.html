About me

Progress and expectations as a Full Stack Developer

I am currently studying in the MIT Full Stack Development program. This course is organized into three main modules: Web Development, Front-End/React Development, and Back-End Development. They consist of the following:

Module 1: Web Development This module is fully completed and in it I gained basic skills in web development using Javascript, HTML and CSS.

In the next months I plan to do the following modules:

Module 2: Front-End Development and React. In it I will dive into front-end development using one of the most popular frameworks, React. I will be expected to be able to write clean and concise code with ES6 JavaScript to be able to work with web components in React. I will learn how to build React applications and how to test and deploy them.

Module 3: Back-End Development Work with MongoDB, ExpressJS and NodeJS to gain confidence in the MERN stack. I will learn the essentials of working in the cloud, automated testing and deployment to get a solid foundation in DevOps.

As you can see this year is a year of great challenges for me, during the whole process I will be adding my projects in this platform, so you can visualize my progress and as I advance in the modules, I will be updating this information.
